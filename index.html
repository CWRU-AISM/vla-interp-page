Abstract

Vision-Language-Action (VLA) models are powerful generalist policies for robotic manipulation, yet how they translate multimodal inputs into motor commands remains opaque. We present the first large-scale mechanistic interpretability study of VLAs, applying activation injection, Sparse Autoencoders (SAEs), and linear probes across six model architectures spanning three orders of magnitude in parameter count: Pi0.5 (3B), OpenVLA-OFT (7B), ACT-ALOHA (80M), GR00T N1.5 (3B), SmolVLA (450M), and X-VLA (0.9B). Our experiments encompass 100,000+ rollout episodes across four simulation benchmarks (LIBERO, SimplerEnv, ALOHA-sim, MetaWorld).

We discover that visual pathway activations dominate robot behavior (cosine similarity = 0.999), while language prompts are largely ignored post-fine-tuning (ANOVA p > 0.24). Per-token SAEs are essential for faithful reconstruction: mean-pooled SAEs cause 88% task failure despite high explained variance. A single SAE trained on mixed data generalizes across four fine-tuned model variants with 99.2% success. Cross-architecture comparison reveals that hidden dimension width determines ablation resilience: narrow models (1024-dim) show catastrophic fragility while wide models (4096-dim) exhibit graceful degradation. We release Action Atlas, an interactive visualization platform for exploring VLA concept representations, alongside the first large-scale dataset of VLA activations and causal intervention results.

Method

Per-token Sparse Autoencoders + causal interventions for VLA mechanistic interpretability

Figure 2: Architecture Comparison. Spider diagram comparing interpretability properties across Pi0.5 (flow matching, dual-pathway) and OpenVLA-OFT (L1 regression, single backbone). The dual-pathway architecture of Pi0.5 enables pathway-specific analysis: PaliGemma encodes WHAT (76% goal, 100% object classification) while the Gemma Expert encodes HOW (R²=0.45 world model, AUC=0.93 success prediction).

Sparse Autoencoders (SAEs)

We train TopK SAEs (k=64) on individual action token activations with 8x expansion (1024 to 8192 for Pi0.5, 4096 to 32768 for OFT). We process each of the 50 action tokens independently, preserving the heterogeneous temporal structure critical for motor control. Mean-pooling across tokens destroys this structure, causing 88% task failure despite high reconstruction quality.

Feature importance is scored via frequency-weighted contrastive selection using Cohen's d effect size between concept-present and concept-absent episodes, multiplied by activation frequency.

Activation Injection & Causal Interventions

We establish causality through four injection conditions: null injection (correct prompt to empty string), same-scene steering (redirect to alternate targets), cross-task injection (transfer across visual scenes), and cross-seed (same task, different initial conditions).

Concept ablation zeros out specific SAE features during live rollouts, while feature steering scales feature activations by alpha to amplify or suppress encoded behaviors.

Figure 3: SAE Explained Variance. Layer-wise analysis of SAE reconstruction quality across all 18 layers for both Expert and PaliGemma pathways, with concept density heatmaps showing how different concepts distribute across the network depth.

Linear Probes

Ridge regression probes achieve 97-98% R² across all action dimensions. We validate causality via a projection operator test: projecting out the probe direction reduces R² to 0%, confirming that the identified directions are causally necessary for action prediction.

Counterfactual Prompting

We systematically test language grounding with 6 prompt variations across 3,396+ episodes: baseline, null (empty string), negation ("don't move"), motor commands, object swap, and temporal switches. The finding that behavior is invariant (p > 0.24) establishes that language is ignored post-fine-tuning.

Figure 4: Temporal Criticality. (A) Feature ablation effects by episode phase: early steps are critical (-49% at step 0), late phases are tolerated (-1%). (B) Linear probes achieve 97-98% R² while SAE ablation shows only 2% effect due to representation redundancy.

Figure 5: The Goldilocks Effect. Unlike LLMs where features smoothly modulate behavior, VLA features show strict sensitivity: any deviation from natural activation levels (boosting or dampening) causes catastrophic failure. VLAs encode precise motor control signals.

Figure 6: Language Is Ignored. (A) Success rates are invariant across 6 prompt types (p > 0.24, 3,396+ episodes). (B) Paradoxically, Layer 17 classifiers achieve 99.3% accuracy distinguishing prompt types, yet behavior is unchanged.

Figure 7: Linear Probes vs SAE Ablation. Per-dimension predictability (R² values) and causality validation via projection. Linear probes identify the exact subspace used for action generation, while SAE ablation is limited by feature redundancy.
